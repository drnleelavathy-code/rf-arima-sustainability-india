# Synthetic Consumer Behavior Dataset
## Hybrid RF-ARIMA Framework for AI-Driven Sustainable Development

**Paper:** A Hybrid Machine Learning Framework for AI-Driven Sustainable Development  
**Journal:** MDPI (Technical Note)  
**Authors:** Ritesh Kalidindi, Leelavathy Narkedamilly, Uma Meghana S  
**Seed:** 42 | **Records:** 10,512 | **Python:** 3.9+

## Dataset Description
Synthetically generated consumer behavior dataset calibrated from
prior empirical literature (Deloitte India 2023, CSE 2024,
Joshi & Rahman 2015). Weights used in Equation (1) of the paper.

## Files
| File | Description |
|---|---|
| `synthetic_consumer_dataset.csv` | Full dataset (10,512 × 21 columns) |
| `dataset_generation_script.py` | Python script (seed=42) |
| `dataset_README.txt` | Full column codebook |
| `MDPI_Supplementary_Dataset.zip` | All files zipped |

## Reproduction
```python
pip install numpy pandas
python dataset_generation_script.py
```
